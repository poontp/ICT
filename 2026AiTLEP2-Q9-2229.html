(b) 完成 findempty(x,y) 的偽代碼

x=
y=
findempty(x,y)
全程變量 e
cnt ← 0
j ← y
當 j >= 0 及 (b1) L[x][j] <> 0
j ← j - 1
當 (b2) j >= 0 及 L[x][j] = 0
j ← j - 1
(b3) cnt ← cnt + 1
e ← [x, (b4) j + 1, (b5) cnt]

動態解析：findempty(4, 5)

(c) 完成 move(x,y) 的偽代碼與執行過程

y=
move(x, y)
全程變量 e
e ← findempty(x, y)
如果 (c1) e[2] > 0
j ← 0
當 (c2) y + j <= 7 及 L[x][y+j] <> 0
(c3) L[x][e[1]+j] ← L[x][y+j]
(c4) L[x][y+j] ← 0
j ← j+1
